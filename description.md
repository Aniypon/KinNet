# Описание проекта

KinNet — веб-приложение для организации семейной жизни. Оно помогает хранить сведения о родственниках, вести семейное древо, планировать события и задачи, обсуждать дела в общем чате, собирать рецепты, вести покупки, списки желаний, цели, семейный альбом, данные о здоровье, опросы и капсулы времени.

## Цель

Создать единое цифровое пространство для семьи, где важные даты, дела, покупки, здоровье и семейная память доступны из браузера на компьютере и телефоне.

## Основные функции

- семейные группы и приглашения участников;
- профили родственников и визуальное семейное древо;
- календарь событий, дни рождения и комментарии;
- задачи с чек-листами, ответственными, бюджетом и вкладами;
- единый раздел «Планы и покупки» для списков покупок, списков желаний и общих целей;
- семейный альбом, где фотография может быть связана с событием или добавлена без привязки;
- рецепты с созданием списка покупок;
- карты здоровья и расписание лекарств;
- опросы, достижения и капсулы времени;
- адаптивный интерфейс для телефонов и режим повышенной читаемости.

## Технологии

- backend: Django, Django Ninja, Celery;
- данные: SQLite для локального запуска, PostgreSQL и Redis в Docker;
- frontend: серверные шаблоны Django, Vite, Tailwind, HTMX, Alpine.js;
- PWA: manifest, service worker и offline-страница.

## Пользователи

Приложение рассчитано на семьи, которым нужен общий центр для планирования, хранения семейной информации и спокойной координации повседневных дел.
